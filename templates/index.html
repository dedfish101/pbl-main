Architecture Note

Tier 1 (Deterministic) — Local Python rules catch high-speed floods and oversized packets instantly with zero API cost or latency.
Tier 2 (Cognitive) — Groq's Llama 3.3 70B model receives the full behavioral fingerprint (IAT, direction ratios, byte volume) and returns a contextual verdict identifying likely intent, risk level, and recommended action. The ai_lock + 3s cooldown prevents API rate-limiting.
